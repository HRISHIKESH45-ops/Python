# IShowSpeed Rock Paper Scissors Game

A text-based Rock-Paper-Scissors game built in Python that reacts using the voice, phrases, and sound effects of IShowSpeed.

## Disclaimer
This project is a lighthearted tribute created entirely for fun. I am a huge fan of IShowSpeed, and there is absolutely no hate intended toward him. This was built spontaneously just as the idea came to mind!

## Warning
**⚠️ PLEASE NOTE: Lower your headphone volume to a safe and comfortable level before playing. IShowSpeed sound effects can be extremely loud and sudden!**

## How to Play
1. Clone or download this repository.
2. Ensure you have your four sound files (`win.mp3`, `lose.mp3`, `tie.mp3`, `exit.mp3`) placed inside the project folder.
3. Run the Python file:
   ```bash
   python Rock_paper_scissor_game.py
   ```

## Requirements
* Python 3.x
* `playsound` library
